# Documentation

This folder contains all project documentation, guides, and hackathon materials.

## 📁 Folder Structure

### `/guides`
Internal development guides and summaries:
- Implementation guides
- Catalog review documentation
- Working prototype summaries
- Migration guides

### `/hackathon`
Hackathon submission materials:
- Video recording guides
- Sample Hindi descriptions
- Submission checklists
- Presentation materials
- Demo links

### `/setup`
Setup and deployment guides:
- Quick start guides
- Demo setup instructions
- Deployment guides
- Mobile app setup

### Root Documentation
Core AWS and technical documentation:
- AWS setup guides
- API documentation
- Service comparisons
- Troubleshooting guides
- Task completion records

## 🚫 Git Ignore

All documentation in `/guides`, `/hackathon`, and `/setup` folders is excluded from git commits to keep the repository clean and focused on code.

## 📝 Note

These files are kept locally for development reference but not pushed to GitHub. Only essential documentation remains in the repository root.
